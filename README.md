# Hydro

An assistant that controls your water intake throughout the day.
Works on Android.

## Advantages of application
- minimalistic UI, 
- basic data processing
- graph and data visualization
- local database
- notifications that remind you to drink water

## Technical aspect
- Flutter and dart
- SQLite
